# Half-Plane Algorithm

Алгоритм обработки системы линейных неравенств  для нахождения вершин допустимой области (выпуклого многоугольника).

## Описание

Проект реализует алгоритм, который:
1. Считывает систему линейных неравенств вида **αx + βy ≤ γ**
2. Удаляет избыточные неравенства (следствия)
3. Находит все вершины допустимой области
4. Визуализирует результат с помощью интерактивных графиков

Запустите основной скрипт визуализации:


Алгоритм автоматически:
- Считывает данные из `halfplane.csv`
- Обрабатывает систему неравенств
- Строит интерактивный график с кнопками переключения

## Как работает алгоритм

### 1. Удаление избыточных неравенств (`buble`)
Находит и удаляет неравенства, которые являются следствием других (избыточные ограничения).

### 2. Сортировка по углу (`vec_numbering`)
Упорядочивает нормальные векторы прямых по углу относительно оси X.

### 3. Поиск вершин
Для каждой пары соседних прямых находится точка пересечения:
- Если координаты целые — вершина добавляется сразу
- Если нет — используется алгоритм из `algo.py` для поиска ближайших допустимых целочисленных точек

### 4. Матричные преобразования (`transform`)
Используются для приведения системы к удобному виду для дальнейшего обхода.

## Формат входных данных

Файл `halfplane.csv` должен содержать три колонки:

| alpha | beta | gamma |
|-------|------|-------|
| 2     | 1    | 10    |
| 1     | 3    | 15    |
| ...   | ...  | ...   |

Каждая строка задаёт неравенство: **αx + βy ≤ γ**

## Что показывает визуализация

График имеет кнопки для переключения между режимами:
- **Старые линии** — все исходные прямые
- **Новые линии** — прямые после удаления избыточных
- **Точки** — найденные вершины многоугольника
